
# Poly Lab Manual Bot

This bot provides Lab Manuals for Polyechnics Diploma students of Kerala from the SITTR site on Telegram using the Aiogram module.


## Bot Link

https://t.me/Poly_Lab_bot

## Running the Bot

To run this bot, type the following command:

```bash
  /start
```
